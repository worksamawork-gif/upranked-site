# Prompt: Social Media Content for upranked.io

Templates for LinkedIn, X (Twitter), and Instagram content.

---

## LinkedIn Post Prompt

```
Write a LinkedIn post for Sama Alaa, founder of upranked.io (GCC SEO agency).

TOPIC: [topic — e.g. why Arabic SEO is underserved, a specific GCC market insight, APEX Framework insight]
SOURCE: [blog post title / client insight / market observation]
LINK: [optional — link to blog post or contact page]

VOICE:
- First person (Sama Alaa speaking)
- Practitioner insight, not agency marketing
- Specific to GCC markets — names cities, sectors, real observations
- No motivational fluff, no generic "hot take" format
- 3-5 short paragraphs, ending with a question or observation that invites comment
- CTA: "Book a free APEX Diagnostic: upranked.io/contact"

LENGTH: 150-300 words
FORMAT: Plain text (LinkedIn doesn't render markdown)
HASHTAGS (3-5 max): #GCCSEO #DubaiSEO #[relevant market] #SEOStrategy
```

---

## X / Twitter Thread Prompt

```
Write a Twitter/X thread for @Upranked7 (upranked.io) on the topic: [TOPIC]

VOICE: Sharp, direct, GCC-market-specific. No generic SEO takes.
FORMAT:
- Tweet 1: Hook (bold claim about GCC market)
- Tweets 2-6: One insight each, numbered
- Tweet 7: CTA → upranked.io/blog/[slug] or upranked.io/contact

MAX 280 chars per tweet. No em dashes.
```

## Instagram Caption Prompt

```
Write an Instagram caption for @upranked.io

POST TYPE: [educational / behind-the-scenes / client result / market insight]
VISUAL: [description of what the image/reel shows]

VOICE: Sama Alaa, premium GCC SEO. Professional but approachable. Arabic/English awareness.
LENGTH: 100-200 words + hashtags
CTA: "Link in bio" or "DM for a free APEX Diagnostic"
HASHTAGS (10-15): mix of #DubaiSEO #GCCSEO #SaudiSEO #DigitalMarketing #SEO #[industry tags]
```
